## 1. 系统概述

### 1.1 产品简介

有序日常是一款集**日程管理、定时提醒、私密生活记录、AI 智能辅助**于一体的轻量化综合生活管理工具。帮你高效管理时间，有序规划生活。

### 1.2 五大核心功能

| 模块 | 功能 | 适用场景 |
|------|------|----------|
| 日历管理 | 月视图日历、日程增删改查、分类标签、优先级标记 | 查看每日安排、快速添加任务 |
| 提醒中心 | 时间轴待办展示、今日/未来/过期分段、晨间推送 | 一目了然查看所有待办 |
| 生活动态 | 仿朋友圈私密记录、图片九宫格、日程关联 | 记录学习/工作/生活点滴 |
| AI 助手 | 自然语言输入生成日程、个人成长简历 | 快速录入、智能整理 |
| 个人中心 | 资料管理、技能标签、AI 成长简历、数据统计 | 查看个人成长轨迹 |

### 1.3 使用方式

- **H5 网页版**：通过手机浏览器访问，无需安装
- **使用前建议注册账号**：注册后可实现数据云端同步，更换设备不丢失
- **游客模式**：无需注册即可体验基本功能，但数据仅存储在本地浏览器

## 2. 快速开始

### 2.1 注册账号

1. 打开应用，点击底部导航栏 **"我的"**
2. 点击 **"登录 / 注册"** 按钮
3. 切换到 **"注册"** 标签页
4. 填写以下信息：
   - **邮箱**（必填）：用于登录和找回密码
   - **密码**（必填）：至少 6 位
   - **昵称**（可选）：不填则自动使用邮箱前缀
   - **个人简介**（可选）：一句话介绍自己
   - **近期目标**（可选）：如"通过期末考试"、"完成毕业设计"
   - **技能标签**（可选）：如"Python, 写作, 摄影"
   - **兴趣爱好**（可选）：如"跑步, 阅读"
5. 点击 **"注册"** 按钮
6. 注册成功后会自动弹出**新用户引导**，介绍五大功能模块

### 2.2 登录账号

1. 进入 **"我的"** → 点击 **"登录 / 注册"**
2. 输入邮箱和密码
3. 点击 **"登录"**
4. 登录后数据会自动从云端同步

### 2.3 底部导航

底部固定栏包含四个标签页：

- **日历**：日历月视图 + 今日待办
- **提醒**：时间轴形式展示所有待办
- **动态**：私密生活记录（朋友圈样式）
- **我的**：个人中心 + AI 成长简历

点击对应图标即可切换。

## 3. 日历管理

### 3.1 日历视图

日历页以**月视图**形式展示最近 30 天的日期格子。每个格子内以**彩色横条**标识当日日程，颜色与分类对应：

- 蓝色（#4A90E2）：学习
- 橙色（#FFA500）：工作
- 绿色（#50C878）：生活

> 每个格子最多显示 **3 个**彩色标识，超出部分显示 "**...**"

### 3.2 切换月份

- 点击左右箭头切换上/下月
- 点击 **"今天"** 按钮快速回到当前日期

### 3.3 添加日程

**方式一：点击日期格子快速添加**

1. 在日历网格中点击任意日期
2. 弹出新增日程表单，日期已自动填充
3. 填写日程信息（详见下方"日程字段说明"）
4. 点击 **"保存"**

**方式二：使用 FAB 按钮**

1. 点击右下角 **蓝色 + 按钮**
2. 填写日程信息
3. 点击 **"保存"**

**方式三：使用 AI 助手**（推荐）

见第 6 章 [AI 智能助手](#6-ai-智能助手)。

### 3.4 日程字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 标题 | 是 | 日程名称，如"计算机网络大作业" |
| 日期 | 是 | 日程日期 |
| 时间 | 是 | 日程时间 |
| 分类 | 否 | 学习/工作/生活 |
| 优先级 | 否 | 低（1）/ 中（2，默认）/ 高（3） |
| 地点 | 否 | 活动地点 |
| 备注 | 否 | 补充说明 |
| 提醒 | 否 | 设置提醒时间 |

### 3.5 编辑日程

1. 在今日待办列表中点击目标日程
2. 弹出编辑表单（已填充现有数据）
3. 修改后点击 **"保存"**

### 3.6 删除日程

1. 在今日待办列表中点击目标日程
2. 在编辑表单底部点击 **"删除"**

### 3.7 标记完成

在日历页底部的今日待办列表中，点击日程左侧的**复选框**，即可标记为完成/未完成。

- 未完成：显示在今日待办列表上方
- 已完成：显示灰色 + 删除线，移至列表底部

## 4. 提醒中心

### 4.1 时间轴布局

提醒中心以**时间轴**形式展示所有待办事项，分为三个区域：

**📌 今日剩余待办**
- 置顶显示，按时间先后排列
- 蓝色时间节点，高优先级任务标红
- 可点击编辑、标记完成

**📅 未来待办**
- 按日期分组展示
- 绿色时间节点
- 可点击编辑或删除

**⏰ 已过期任务**
- 自动置灰并移至底部
- 灰色时间节点
- 仍然可以编辑或删除

### 4.2 每日温馨卡片

提醒页顶部展示每日问候语和数据统计：
- 今日待办数量
- 未来 7 天待办数量
- 动态变化（根据每日任务量调整问候语）

### 4.3 晨间推送设置

提醒页底部可设置每日晨间待办推送：

1. 打开/关闭 **"晨间推送"** 开关
2. 点击时间选择器设置推送时间（默认 08:00）
3. 到达设定时间时，可在通知栏看到当日待办清单

> **注意**：H5 网页版系统通知功能受限，建议在原生 APP 中使用。

### 4.4 已完成任务显示

已完成的任务会以以下样式展示：
- 文字添加**删除线**（line-through）
- 降低不透明度（opacity: 0.6）
- 仍可编辑和删除

## 5. 生活动态

### 5.1 动态列表

动态页采用**仿微信朋友圈**布局设计，完全私密，仅自己可见。

每条动态卡片包含：
- **左侧日期栏**：显示日期（MM-DD）和中文星期
- **头像 + 昵称**：微信蓝色昵称
- **相对时间**：刚刚 / X分钟前 / X小时前 / 昨天 / 具体日期
- **正文内容**：支持长文本，超过 6 行自动折叠
- **图片九宫格**：支持 1-9 张图片
- **关联日程链接**：如有关联日程，显示蓝色链接卡片

### 5.2 发布动态

1. 点击右下角**微信绿色 FAB 按钮**（带编辑图标）
2. 在弹出的发布表单中：
   - 输入**文字内容**（必填）
   - 点击 **"添加图片"** 选择照片（可选）
   - 点击 **"关联日程"** 从最近 7 天日程中选择（可选）
   - 选择**日期**（默认为今天）
3. 点击 **"发布"**

### 5.3 长文本折叠

- 动态正文超过 **140 字符**（约 6 行）时自动折叠
- 点击 **"全文"** 展开完整内容
- 再次点击 **"收起"** 恢复折叠

### 5.4 图片预览

- 点击任意图片可**全屏预览**
- 预览模式下可**左右滑动**查看其他图片
- 点击空白区域退出预览

### 5.5 关联日程

动态可以关联到具体的日程事项：
- 发布动态时点击"关联日程"选择对应日程
- 发布后动态卡片显示关联日程的蓝色链接
- 例如：完成"计算机网络大作业"后发布动态，关联该日程

### 5.6 编辑/删除动态

- 直接在动态卡片上点按可查看详情
- 在详情中可编辑或删除

## 6. AI 智能助手

### 6.1 进入 AI 助手

点击日历页右下角的 **✨ 按钮**（浮在蓝色 FAB 上方），打开 AI 对话窗口。

### 6.2 文字输入生成日程

直接用自然语言告诉 AI 你想做什么：

**示例 1**：`明天下午3点去图书馆复习计算机网络`

AI 会自动解析：
- 日期：明天（自动计算具体日期）
- 时间：15:00
- 标题：去图书馆复习计算机网络
- 地点：图书馆

**示例 2**：`下周五晚上7点在会议室开项目讨论会`

**示例 3**：`6月20日上午9点提交论文初稿，优先级高`

### 6.3 AI 支持的日期表达

AI 能理解以下中文日期和时间表达：

| 表达方式 | 示例 |
|----------|------|
| 相对日期 | 今天、明天、后天 |
| 星期表达 | 周一、下周三、周五 |
| 具体日期 | 6月20日、2026-6-20 |
| 时间段 | 上午、下午、晚上、中午 |
| 具体时间 | 下午3点、15:00、晚上8点 |

### 6.4 日程预览与确认

1. AI 解析完你的描述后，会生成**日程预览卡片**
2. 检查标题、日期、时间、地点是否正确
3. 点击 **"添加到日历"** 确认添加
4. 如需修改，点击 **"放弃"**，然后用文字告诉 AI 需要调整的地方

### 6.5 AI 降级方案

当 AI 服务不可用时，系统会自动切换为**本地规则引擎**：
- 仍能识别大部分常用日期和时间表达
- 如果信息不完整，会友好地追问你
- 功能体验基本一致

### 6.6 语音输入（H5 版）

点击 AI 聊天框下方的**麦克风按钮**，使用浏览器语音识别输入：

1. 首次使用需授权浏览器麦克风权限
2. 点击麦克风开始录音
3. 说出你的日程描述
4. 识别完成后自动填入输入框
5. 点击发送按钮提交给 AI

> H5 版使用 Web Speech API，需要 Chrome 或 Edge 浏览器支持。

## 7. 个人中心

### 7.1 个人资料

进入 **"我的"**，在头部区域可以：
- 查看头像、昵称、欢迎语
- 点击 **"编辑资料"** 修改：
  - 头像（点击上传）
  - 昵称
  - 个人简介
  - 近期目标
  - 技能标签
  - 兴趣爱好

### 7.2 数据统计

在个人中心头部下方，展示两个核心数据：
- **完成日程数**：已标记完成的日程总数
- **发布动态数**：已发布的动态总数

### 7.3 AI 成长简历

登录后，系统根据你的数据自动生成个性化成长简历（由数据驱动，非虚构）：

**简历卡片包含**：

**📝 个人概述**
- 基于你填写的个人简介和目标生成的自我介绍
- 橙色左边框卡片样式

**🏷️ 技能标签云**
- 从你填写的技能 + 已完成日程自动提取
- 蓝色药丸样式标签，最多显示 8 个
- 例如：完成编程相关日程 → 自动添加"编程"标签

**⭐ 亮点时刻**
- 展示最近的已完成日程（里程碑事件）
- 时间轴样式 + 蓝色节点
- 例如："2026-06-15 完成计算机网络大作业"

**🌱 成长洞察**
- 根据完成数量生成的鼓励文案
- 绿色左边框卡片样式
- 完成 ≥ 10 项："执行力令人印象深刻！"
- 完成 5-9 项："每天的行动积累正在让目标变得更近"
- 完成 1-4 项："已踏出第一步，每一步都算数"
- 0 项："不妨从现在开始，设定一个小目标"

**💡 下一步建议**
- 基于你填写的目标生成行动建议
- 编号列表样式
- 例如："朝着「通过期末考试」的目标再迈一步"

**🔄 重新生成**
- 点击底部"重新生成"按钮刷新简历内容

### 7.4 游客状态

未登录时，个人中心显示：
- 引导卡片："登录后解锁个人成长简历"
- **登录** 和 **注册** 按钮

### 7.5 设置菜单

登录后底部显示设置菜单项：

| 菜单项 | 状态 |
|--------|------|
| 分类管理 | 即将上线 |
| 提醒与推送设置 | 即将上线 |
| 数据同步 | 即将上线 |
| 通用设置 | 即将上线 |

### 7.6 关于

点击"关于有序日常"查看版本号和产品简介。

### 7.7 退出登录

点击底部 **"退出登录"**，确认后退出。退出后数据保留在手机/浏览器本地，重新登录后自动同步。

### 7.8 账号注销

> ⚠️ **警告**：注销后所有数据将被永久删除，无法恢复！

1. 点击底部 **"账号注销"**（红色文字）
2. 在弹出的确认窗口中输入 **"确认注销"** 四个字
3. 点击 **"确认注销"** 按钮
4. 服务器永久删除账号及所有关联数据（日程、动态）
5. 自动退出登录

## 8. 常见问题 (FAQ)

### Q1：游客模式和注册用户有什么区别？

| 特性 | 游客 | 注册用户 |
|------|------|----------|
| 添加日程 | ✅ | ✅ |
| 查看日历 | ✅ | ✅ |
| 发布动态 | ✅ | ✅ |
| AI 日程生成 | ✅ | ✅ |
| AI 成长简历 | ❌ | ✅ |
| 数据云端同步 | ❌ | ✅ |
| 更换设备保留数据 | ❌ | ✅ |
| 账号注销 | ❌ | ✅ |

### Q2：数据会丢失吗？

- **注册用户**：数据实时同步到云端 MySQL 数据库，更换设备登录后自动恢复
- **游客**：数据存储在浏览器 localStorage，清除浏览器数据会导致数据丢失，建议注册账号

### Q3：AI 生成日程不准怎么办？

1. 尝试用更明确的语言描述，例如加上具体日期和时间
2. AI 会在信息不完整时追问，按提示补充即可
3. 可以在 AI 生成的预览中手动修改
4. AI 服务不可用时，系统会自动切换为本地规则引擎，仍能识别大部分常用表达

### Q4：如何设置日程提醒？

编辑日程时，可设置提醒时间。提醒功能在 H5 网页版受浏览器限制，完整提醒体验建议使用原生 APP。

### Q5：动态的图片为什么会加载失败？

已修复（v2.2）。如果仍遇到，尝试刷新页面。发布动态时上传的图片会自动转换为永久链接保存。

### Q6：支持多设备同步吗？

支持。注册用户登录后，不同设备访问同一地址即可看到相同数据。

### Q7：为什么点击按钮没反应？

v2.3 已修复此问题。如果仍遇到，请尝试刷新页面。

### Q8：如何联系技术支持？

通过项目仓库提交 Issue 或联系开发者。

### Q9："确认注销"输入什么？

在弹出的确认窗口中，输入**中文"确认注销"四个字**（不含引号），然后点击确认按钮。

### Q10：支持修改密码吗？

当前版本暂不支持自助修改密码。该功能计划在后续版本中上线。
