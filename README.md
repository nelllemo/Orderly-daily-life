# 有序日常 (Orderly Daily Life)

一款基于 uni-app 的日常计划与生活记录原型，聚焦日历任务、提醒时间轴、生活动态与个人中心四大模块。

## 功能概览

- 日历视图：月视图日期网格、今日待办、分类标签与优先级提示
- 提醒中心：时间轴展示今日剩余、未来待办与已过期任务
- 生活动态：按日期筛选的动态时间线，支持发布与关联日程
- 个人中心：完成统计、动态统计与常用设置入口
- AI 日程助手（模拟）：输入口语化描述，生成结构化日程

## 说明

- 当前为 UI/交互原型与本地模拟数据，未接入真实后端或 AI 服务
- 主要界面与逻辑集中在 `pages/index/index.vue`

## 运行方式

- 使用 HBuilderX 打开项目
- 通过“运行到浏览器/小程序/手机”进行预览

## 目录结构（节选）

- pages/index/index.vue：主界面与业务逻辑
- components/：原拆分组件（当前已整合到首页）
- static/：静态资源目录
